Amazon Signal Map Skills

Two Claude Code skills that turn a raw Amazon data export into seller intelligence artifacts.

Before you install: You need Claude Code installed. These are skill files — they're instructions that Claude Code reads to run structured workflows.
Option A — One-command install
install.sh — Automatic installer
Downloads and writes all skill files to ~/.claude/skills/ in one shot. Requires Python 3 (ships with macOS).
bash ~/Downloads/install.sh
Option B — Download individual skill files
Skill 1: amazon-data-export-intelligence
Triggers when you have an Amazon export ZIP
Unpacks the archive, inventories all files, classifies high-value datasets, interprets schemas, and produces 11 intelligence artifacts. Phases 1–6, confidence discipline enforced.
Save to: ~/.claude/skills/amazon-data-export-intelligence/SKILL.md
Skill 2: seller-intelligence-packager
Triggers when analysis is done, user wants a deck or playbook
Packages completed analysis artifacts into a presentation deck (14 slides + HTML) or operator playbook (12 sections + HTML). Preserves confidence labels throughout.
Save to: ~/.claude/skills/seller-intelligence-packager/SKILL.md
Option C — Full archive
amazon-signal-map-skills.zip — Complete bundle
Includes both SKILL.md files, CLAUDE.md workspace config, amazon-export-field-reference.md, inspection scripts (inspect-archive.sh + sample-csv.sh), INSTALL.md, and EVAL.md.
# After downloading and unzipping: cp -r amazon-signal-map-skills/amazon-data-export-intelligence ~/.claude/skills/ cp -r amazon-signal-map-skills/seller-intelligence-packager ~/.claude/skills/
Install sequence
1
Download install.sh and run it
bash ~/Downloads/install.sh
Creates ~/.claude/skills/amazon-data-export-intelligence/ and ~/.claude/skills/seller-intelligence-packager/ automatically.
2
Verify installation
ls ~/.claude/skills/ | grep -E "amazon|seller"
Should show both skill folders.
3
Open Claude Code in your analysis folder
cd ~/Desktop/your-analysis-workspace claude
4
Start the workflow
Type /amazon-data-export-intelligence or just say: "I just downloaded my Amazon data export. Let's analyze it."
Which skill to use when
IF
You have an Amazon export ZIP and haven't analyzed it yet
→
Use /amazon-data-export-intelligence
IF
You have SELLER_INTELLIGENCE_REPORT.md and want a deck or playbook
→
Use /seller-intelligence-packager
IF
You're mid-analysis and want to continue from where you left off
→
Say "Let's continue the analysis" — Claude reads existing artifacts automatically
IF
You have a huge export (300+ files) and aren't sure where to start
→
Skill 1 handles this — it triages by seller relevance, not alphabetically
IF
You don't have a ZIP yet but want to understand what the data reveals
→
Request your data at amazon.com/privacy → Privacy & Security → Request your data
